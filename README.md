### Install
```bash
$ npm install --global @asyncapi/cli
```

### CLI
```
$ asyncapi --help

Usage
  $ asyncapi command options
  
Commands
  validate 
   Options
     -c --context  Your AsyncAPI specification
     -w --watch Enable watchMode

Examples
  $ asyncapi validate --context=specification.yml
```
